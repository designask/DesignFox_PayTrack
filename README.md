# DesignFox PayTrack - POS & Business Workflow System

A modern, static POS and business workflow management system for service-based companies. Runs entirely in the browser using localStorage - no server or database required!

## Live Demo

🔗 **https://designask.github.io/DesignFox_PayTrack/**

## Features

### Complete Client Payment Workflow
```
Quotation Approved → Advance Invoice Sent → 50% Payment Received →
Work Starts → Project Completed → Final Invoice Sent →
Balance Payment Received → Final Delivery
```

### Modules
- **Dashboard** - Business stats overview
- **Customers** - Add, edit, search, delete customers
- **Quotations** - Create quotations with multiple service items, approve/reject
- **Invoices** - Advance & final invoices, auto-generated from quotations
- **Payments** - Record payments, auto-generate receipts, multiple methods
- **Projects** - Visual workflow timeline with step-by-step tracking
- **Reports** - Revenue, pending payments, customer summary, conversion rate

### Payment Methods
- Cash
- Bank Transfer
- Card
- Online Payment

## Technology
- Pure HTML5, CSS3, JavaScript
- No frameworks or build tools needed
- Data stored in browser localStorage
- Hosted free on GitHub Pages

## How to Use
1. Login with demo credentials
2. Add customers
3. Create quotations → Send → Approve
4. System creates project & workflow automatically
5. Create invoice from approved quotation (50% advance)
6. Record payment → Receipt auto-generated
7. Track project progress through workflow steps
